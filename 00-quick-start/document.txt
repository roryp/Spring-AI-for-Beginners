Spring AI - Java Framework for AI Applications

Spring AI is a framework from the Spring ecosystem designed to simplify the development of AI-powered applications in Java. It provides a consistent, Spring-friendly API for integrating with various AI model providers including OpenAI, Microsoft Foundry, and GitHub Models.

Key Features:
- ChatModel API: A unified interface for chat completions across providers, with both synchronous and streaming support
- OpenAI SDK Integration: Built on the official OpenAI Java SDK for robust, officially-maintained provider support
- Prompt & PromptTemplate: Structured prompt abstractions with variable substitution using {placeholder} syntax
- Tool Calling: Register Java functions as tools via FunctionToolCallback so the AI can invoke them automatically
- Structured Output: Convert AI responses directly into Java records and objects using JSON Schema
- RAG Support: Retrieval-Augmented Generation with vector stores, document readers, and text splitters
- Advisors: Interceptor-like components for cross-cutting concerns such as logging, guardrails, and memory

Architecture:
Spring AI follows Spring's modular design. The core module (spring-ai-model) defines abstractions like ChatModel, Prompt, and ChatResponse. Provider modules like spring-ai-openai-sdk implement these abstractions using provider-specific SDKs. Auto-configuration modules provide Spring Boot integration with application.properties-based setup.

Use Cases:
- Chatbots and conversational interfaces
- Document question-answering systems
- AI-powered code generation and analysis
- Content summarization and extraction
- Semantic search with vector embeddings
- Agentic workflows with tool calling

Getting Started:
To use Spring AI, add the spring-ai-openai-sdk dependency and the Spring AI BOM to your Maven or Gradle project. Configure the model with your API credentials via environment variables (OPENAI_BASE_URL, OPENAI_API_KEY) or Spring properties (spring.ai.openai-sdk.*), and start building AI-powered features.

GitHub Models Integration:
Spring AI works seamlessly with GitHub Models through the official OpenAI Java SDK. Set the base URL to https://models.github.ai/inference and provide your GitHub Personal Access Token. This provides free access to state-of-the-art models for learning and development purposes without requiring an Azure subscription.
